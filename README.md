# Pinnacle Clinical: Virtual Autism Evaluations

A lightning-fast, high-converting marketing website for **Pinnacle Clinical** (formerly On-Target), focused on delivering Virtual Autism Testing in Utah. The site is engineered to replace a fragmented intake experience with a seamless, mobile-optimized funnel.

## 🎯 Project Goals

*   **Conversion Rate Optimization:** Clear CTAs, transparent insurance lists (Medicaid, Cigna, Aetna, etc.), and immediate waitlist availability messaging.
*   **Search Engine Optimization (SEO):** Deeply optimized with Schema.org JSON-LD, canonical tags, and Open Graph meta properties targeting the "Murray, UT" and statewide telehealth demographics.
*   **Performance:** A single-page architecture built entirely with Vanilla JavaScript and Tailwind CSS for instant load times—crucial for parents searching for care on mobile devices.

## 🏗️ Architecture

Instead of dealing with WordPress bloat, this platform relies entirely on fundamental web primitives:
*   **Styling:** Utility-first CSS via a pre-compiled Tailwind configuration.
*   **Interactivity:** Lightweight Vanilla JS (`script.js`) powering smooth scrolling, mobile navigation menus, and FAQ accordions.
*   **Assets:** Optimized images located in the `/images/` directory to ensure sub-second First Contentful Paint (FCP).

## 🚀 Deployment

The site is production-ready for any static hosting provider (e.g., Cloudflare Pages, Netlify, Vercel). Simply deploy the root directory to serve `index.html`.
